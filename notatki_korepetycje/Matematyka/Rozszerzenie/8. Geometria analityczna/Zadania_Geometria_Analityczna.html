Skategoryzowane Schematy i CKE Karta Wzorów (Klucze)

Karta Wzorów (niezbędnik) :
[1] Długość wektora / odcinka: |AB| = √[(x_B - x_A)² + (y_B - y_A)²]
[2] Środek odcinka: S = ( (x_A + x_B)/2, (y_A + y_B)/2 )
[3] Iloczyn skalarny (prostopadłość): u ∘ v = u_x*v_x + u_y*v_y = 0
[4] Odległość punktu P(x_0, y_0) od prostej Ax + By + C = 0: d = |A*x_0 + B*y_0 + C| / √(A² + B²)
[5] Pole równoległoboku na wektorach u i v (wyznacznik): P = |det(u, v)| = |u_x*v_y - u_y*v_x|
[6] Pole trójkąta na wektorach: ½|det(u, v)|
[7] Tangens kąta między prostymi: tg α = | (a_1 - a_2)/(1 + a_1*a_2) |
[8] Odległość prostych równoległych Ax + By + C_1=0 i Ax + By + C_2=0: d = |C_1 - C_2| / √(A² + B²)
[9] Okrąg o środku S(a,b) i r>0: (x - a)² + (y - b)² = r²
1. Wektory: Równania [Wzór 1]
Z równania: 2w = 3b - a i obliczenia 3b: 3*[-2, 4] = [-6, 12].
Odejmujemy wektor a: 3b - a = [-6 - 3, 12 - (-1)] = [-9, 13].
Dzielimy przez 2: w = [-4.5, 6.5].
Długość |w| = √((-4.5)² + 6.5²) = √(20.25 + 42.25) = √62.5 = 2.5√10 (lub (5/2)√10).
2. Prostopadłość wektorów z parametrem [Wzór 3]
Mnożymy u ∘ v równe 0: (m-1)(m+1) + 3(-4) = 0.
Ze wzorów skróconego mnożenia: m² - 1 - 12 = 0 → m² = 13.
Wynik: m = √13 LUB m = -√13.
3. Pole równoległoboku wektorowego [Wzór 5]
Korzystamy z wyznacznika wektorów u i w: u_x * w_y - u_y * w_x = 2*m - (-3)*3 = 2m + 9.
Moduł wyznacznika jest polem: P = |2m + 9| = 13.
|2m + 9| na dwa równania: 2m + 9 = 13 (2m = 4 → m = 2) LUB 2m + 9 = -13 (2m = -22 → m = -11).
4. Symetralna prostej (AB) [Wzór 2]
Obliczamy środek odcinka S = ((-1+3)/2, (5-3)/2) = (1, 1).
Współczynnik prostej AB: a_AB = (-3 - 5)/(3 - (-1)) = -8 / 4 = -2.
Współczynnik symetralnej z warunku prostopadłości (-1/a_AB): a_sym = 1/2.
Równanie w postaci y = (1/2)x + b, przechodzi przez (1,1): 1 = 1/2*1 + b → b = 0.5. Ostatecznie y = 0.5x + 0.5.
5. Wyznaczanie punktu Q w symetrii [Wzór 2]
Dana jest symetralna l: y = -2x + 7. Prosta k jest do niej prostopadła i przechodzi przez P(4, 5).
a_k = 1/2. Prosta k: y - 5 = 1/2(x - 4) → y = 0.5x + 3.
Znajdujemy punkt przecięcia S obu prostych: -2x + 7 = 0.5x + 3 → 2.5x = 4 → x = 1.6, a y = 0.5(1.6)+3 = 3.8. S(1.6, 3.8).
Z symetrii, wektor PS to wektor SQ. Z średniej podziału (Wzór 2): S = (P+Q)/2 → Q = 2S - P. (3.2 - 4 , 7.6 - 5) = (-0.8, 2.6). Q=(-0.8, 2.6).
6. Wysokość D z wierzchołka [Wzór 4]
Wyznaczamy równanie prostej AB przechodzącej przez (2,1) i (6,5): współczynnik a = (5-1)/(6-2) = 1. Równanie y - 1 = 1(x - 2) → y = x - 1 → x - y - 1 = 0.
Odl. punktu C(-2, 7) od tej prostej to: h = |1(-2) - 1(7) - 1| / √(1² + (-1)²) = |-10|/√2 = 10/√2 = 5√2.
7. Dwie proste równoległe (odległość) [Wzór 8]
l: 3x - 4y + 2 = 0. Prostopadła jest taka sama dla prostych równoległych: k: 3x - 4y + C = 0.
Ze wzoru na odl. prostych: d = |C - 2| / √(3² + (-4)²) = |C - 2| / 5.
I mamy warunek d = 5: |C - 2| / 5 = 5 → |C - 2| = 25. Daje to C = 27 (bo 27-2=25) lub C = -23 (bo -23-2 = -25).
Rozwiązania to proste: 3x - 4y + 27 = 0 LUB 3x - 4y - 23 = 0.
8. Symetria środkowa okręgu [Wzór 9]
Poprzedni środek to S_O1 = (2, -3). Symetria jest względem punktu S = (-1, 1).
Znajdujemy odbicie S_O2 (obraz). S to środek odcinka S_O1--S_O2.
S_O2 = 2*S - S_O1 = 2*(-1, 1) - (2, -3) = (-2, 2) - (2, -3) = (-4, 5).
W symetrii promień t=4 się nie zmienia, nowe równanie to (x + 4)² + (y - 5)² = 16.
9. Tangens kąta między prostymi [Wzór 7]
Szybkal kontrola dla pierwszej poprawności to sprawdzić P(-1,-5) z y=3x-2: -5 = 3(-1)-2. Działa.
Mamy a1=3 i a2=a. Ze wzoru 7: tg α = | (3 - a) / (1 + 3a) | = 1/2.
Zdejmujemy dwustronny moduł: 6 - 2a = 1 + 3a (-5a = -5 → a = 1) oraz 6 - 2a = -1 - 3a (a = -7).
Wyznaczmy b z f faktu, ze P(-1,-5) należy do tych prostych.
Dla a=1: y = x+b → -5 = -1 + b → b = -4. Dla a=-7: y = -7x+b → -5 = 7 + b → b = -12.
To dwie szukane proste (y=x-4 i y=-7x-12).
10. Przecięcie paraboli i odległość [Wzór 1]
Rozwiązujemy elementarny układ równań y = 2x - 1 i y = x² - x + 1. Wstawiamy do góry: 2x - 1 = x² - x + 1 → x² - 3x + 2 = 0.
Ze wzorów Vietea widzimy, że pierwiastki to 1 * 2 = 2 i 1 + 2 = 3. x_1 = 1, x_2 = 2.
Mamy punkty A(1, 1) Oraz B(2, 3). Moduł długości wektora to √((2-1)² + (3-1)²) = √(1+4) = √5.
11. Pole kwadratu z długości przekątnej [Wzór 4]
Sprawdzamy że A(1,1) nie należy do przekątnej y=2x-6 (co upraszcza działanie - punkt jest na drugiej niepowiązanej osi). Przekątna l jako 2x - y - 6 = 0.
Odległość wierzchołka A od l, to odległość od połówki przekątnej do środka czyli R. Ale z własności kwadratu to h=half of D.
D/2 = |2(1) - 1(1) - 6| / √5 = |-5|/√5 = √5.
Skoro D/2 jest √5, to pełna przekątnia D = 2√5. Pole z przekątnej: P = 1/2 * D² = 1/2 * (2√5)² = 1/2 * 20 = 10.
12. Pole trójkąta z parametrem [Wzór 6]
Rozpisz wektor dla prostej ABC. Mamy AB = [3 - (-4), -1 - 0] = [7, -1].
Punkt C leży na prostej y=-2x+4, przyjmijmy sobie C(x, -2x+4). Tworzymy drugi wektor AC = [x+4, -2x+4 - 0] = [x+4, -2x+4].
Wzór z modułu z det: Pole = 1/2 * | 7*(-2x+4) - (-1)*(x+4) | = 1/2 * |-14x + 28 + x + 4| = 1/2 * |-13x + 32|.
Wiemy, że równa się on 17.5. |-13x + 32| = 35. Stąd dwa warianty:
13x = -3 → x = -3/13 (wtedy y = 58/13)
13x = 67 → x = 67/13 (wtedy y = -82/13). Dwa prawidłowe punkty pod wyznacznik.
13. Środek jednokładności k > 0
O1 ma S_1(0,0) i r_1=2. O2 ma S_2(3,4) i r_2=4. O2 jest wynikiem jednokładności (bo O1 jest jednokładny do O2), zatem skala k = r2/r1 = 4/2 = 2. Liczba dodatnia zachowuje ukierunkowanie wektora ze środka J.
J musi leżeć na prostej łączącej środki okręgów S_1 S_2.
Tworzymy własność wektorową jednokładności: Wektor (JS_2) = 2 * (JS_1). (J jako nasza niewiadoma wsp).
Dla x: 3 - x = 2*(0 - x) → 3 - x = -2x → x = -3.
Dla y: 4 - y = 2*(0 - y) → 4 - y = -2y → y = -4.
Środek jednokładności znajduje się w S(-3, -4).
14. Pełne Równanie Stycznej Okręgu [Wzór 4 i 9]
Promień to dystans od środka S(3,2) do prostej stycznej l(2x-y-1=0).
R = |2(3) - 2 - 1| / √(2²+(-1)²) = |3|/√5 = 3/√5. A więc r² = 9/5 = 1.8.
Równanie kanoniczne i redukcja do ogólnego: (x-3)² + (y-2)² = 1.8.
x² - 6x + 9 + y² - 4y + 4 - 1.8 = 0 → x² + y² - 6x - 4y + 11.2 = 0.
15. Wierzchołki z pola rombu i symetralnej [Rygor Opracowania CKE]
Przekątna d_1 = |AC| = √(3² + 4²) = 5. Skoro Pole=25 a 1/2*d_1*d_2 = 25 → d_2 = 10. To oznacza, że nowa przekątna jest 2x dluższa. B i D są oddalone od środka S o wartość 5 (10/2).
S(srodek AC) = (2.5, 0). Symetralna prostej AC przechodzi przez (2.5,0) i jest prostopadła do AC (a_ac = 4/3). Czyli a_sym = -3/4. Wynik symetralnej: y - 0 = -3/4(x - 2.5) → y = -0.75x + 1.875.
Wiemy, że wierzchołki leżą na okręgu z promieniem r=5 (środek w 2.5, 0): (x-2.5)² + y² = 25. Rozwiązujemy układ wstawiając y: (x-2.5)² + (-3/4(x-2.5))² = 25 → (1 + 9/16)(x-2.5)² = 25 → 25/16 (x-2.5)² = 25 → (x-2.5)² = 16.
Czyli x-2.5 = 4 LUB x-2.5 = -4.
To daje punkty x=6.5 z y=-3 oraz x=-1.5 z y=3.
16. Okrąg Trójkąta Równobocznego
Bok trójkąta a = |AB| = √(2²+(-2)²) = 2√2. Zatem wysokość h = a√3/2 = (2√2 * √3)/2 = √6.
W równobocznym promień opcisanego R to (2/3)h = (2/3)√6. Zatem R² = (4/9)*6 = 8/3.
Środek leży na symetralnej odcinka podziału AB. Środek S_AB = (4,1). Wektor kierunku AB ma wsp a = -2/2 = -1 (równanie y = -x+5). Symetralna to y = x-3.
Używa się znanej odległości S do Boku h/3 aby utorować wektor. S(x, y) na y=x-3 spełnia (x-4)² + (y-1)² = (h/3)². Ponieważ (h/3)² = (1/9)*6 = 2/3.
(x-4)² + (x-4)² = 2/3 → 2(x-4)² = 2/3 → (x-4)² = 1/3 → x = 4 ± √3/3.
Daje to dwa możliwe środki: S1 = (4+√3/3, 1+√3/3) oraz S2 = (4-√3/3, 1-√3/3). Z promieni r²=8/3 dają dwa równania kanonicze okręgu!
17. Parametr prostej z okręgiem y=x
Aby równanie x² + y² - 4x + 6y + k = 0 zdefiniowało Rzeczywisty Okrąg, zwijamy to do: (x-2)² + (y+3)² = 13 - k. Upewniamy maturalnie się, że promień istnieje: R² = 13 - k > 0 → k < 13.
Z linii y=x budujemy układ: x² + x² - 4x + 6x + k = 0 → 2x² + 2x + k = 0.
By miały *dokładnie dwa* punkty złączne Δ > 0: 4 - 8k > 0 → 8k < 4 → k < 1/2.
Warunek podstawowy i okręgu konfirmowany → k ∈ (-∞, 0.5).
18. Środek Na OX wpisania / odległość punktowa
Brak przeciwprostokątnej AB wymusza sytuację w której C tworzy przeciwległy biegły, a środek S jest zawsze pośrodku hipotenuz dla prostokątnych trójkątów. Środek na OX mówi że y_s = 0; S(x_s, 0).
Punkty okręgu leżą na promieniu wię |SA|² = |SB|². Równanie: (2-x_S)² + (-3-0)² = (-4-x_S)² + (5-0)².
Rozpisane: 4 - 4x_S + x_S² + 9 = 16 + 8x_S + x_S² + 25. Kasujemy kwadraty. 13 - 4x_S = 41 + 8x_S → -12x_S = 28 → x_S = -28/12 = -7/3.
Masz środek! Teraz R² wyliczane z (2 - (-7/3))² + 9 = (13/3)² + 9 = 169/9 + 81/9 = 250/9. Podstawić do równania! O: (x+7/3)²+y² = 250/9.
19. Okrąg Na Przyprostokątnej i wyznaczanie dopełnień [Schemat Ostateczny]
O: (x-1)² + (y+2)² = 25 posiada S=(1,-2). Spradzenie A(4,2) → (4-1)² + (2+2)² = 25. Istotnie leży. Kąty proste w okręgu opierają się na średnicy! Średnica okręgu to Czasami przeciwprostokątna.
Pierwsza przyprostokątna prostej zawiera linię x-2y=0 (więc y=0.5x). B musi zatem leżeć na okręgu ORAZ prostej. Podstawiamy prosta do okręgu:
(2y-1)² + (y+2)² = 25 → 4y² - 4y + 1 + y² + 4y + 4 = 25 → 5y² = 20 → y² = 4. Wiec y = -2 lub y = 2.
Dla y=2, x=4 (To nasz znany punkt A). Dla y=-2, x=-4. To nasz wirtualny pierwszy punkt boczny B(-4, -2).
Drugi bok od A musi być do niego PROSTOPADŁY i na Nim leży wierzchołek C okręgowy! Z y=0.5x bierzemy współczynnik -2. z A(4,2): y - 2 = -2(x - 4) → druga krzywa to y = -2x + 10.
Podpinamy y=-2x+10 na okrąg: (x-1)² + (-2x+10+2)² = 25 → x²-2x+1 + 4x²-48x+144 = 25 → 5x² - 50x + 120 = 0 → x² - 10x + 24 = 0.
Mamy pierwiastki x=4 (czyli nasze punkt A, z logiki) oraz x=6. Dla x=6 y = -2(6)+10 = -2. Wierzchołek C jest w (6, -2).
20. Trzeci Wierzchołek Równobocznego C i Dystans Moduł
S_AB = (5, 1). A=(3,-1) B=(7,3). |AB| = √(4² + 4²) = √32 = 4√2. Dla równobocznego h = a√3/2 = 4√2 * √3 / 2 = 2√6.
Symetralna AB o wsp a=1 wprowadzona przoz srodek (5,1) ma równanie y = -x + 6.
Względem długości, C może lądować w obie strony po 2√6 od punktu centralnego. Wstawiamy z symetralnej by rozwiązać (x_C - 5)² + (y_C - 1)² = (2√6)².
(x_C - 5)² + (-x_C + 6 - 1)² = 24 → 2(x_C - 5)² = 24 → (x_C - 5)² = 12 → x_C - 5 = ±√12 = ±2√3.

22. Punkt Maksymalizujący Pole O Partycji na Okręgu O (Optymalizacja Modułu Ekstremum)
Punkt O(1, 3). Pole P(ABC) opiera się na bazie AB (tu (2,0) i (4,2)) i max wysokości C od AB. Baza |AB| = √((4-2)²+(2-0)²) = √8 = 2√2.
Przecinająca prosta AB to y=x-2 → x - y - 2 = 0.
Punkt na okręgu leżący najdalej od stycznej linii to przecięcie tej funkcji z prostą centralną przechodzącą przez centrum okręgu O(1,3).
Jako że AB ma wsp. 1 to prostopadła -1 (przechodząca przez (1,3): y - 3 = -1(x-1) → y = -x+4.
Punkty okręgowe z y=-x+4: (x-1)² + (-x+4-3)²=10 → (x-1)²+(-x+1)²=10 → 2(x-1)²=10 → (x-1)²=5. Oznacza to, że x można brać jako 1+√5 albo 1-√5.
Z C1 = (1 - √5, 3 + √5) liczysz odległość od x - y - 2 = 0. Jej moduł bezwzględny |-4 - 2√5| jest NAJWIĘKSZY, na czym nam optymalizacyjnie zależy.
Odległość d = (4+2√5)/√2 = 2√2 + √10.
Finalna formułą to P = 1/2 * (Baza) * d(max) = 1/2 * 2√2 * ((4+2√5)/√2) = 4 + 2√5. Podany punkt i konkretna max-wielkośċ spełnia wymogi zadania.
