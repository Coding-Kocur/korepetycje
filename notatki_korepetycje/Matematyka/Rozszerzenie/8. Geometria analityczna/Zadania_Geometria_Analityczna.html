Zadanie 12. CKE Maj 2015

Schematy rozwiązań dla nauczyciela

Zadanie 1. (Symetralna odcinka)
1. Wyznacz środek odcinka AB: S = ( (xA+xB)/2, (yA+yB)/2 ) = (1, 1).
2. Wyznacz współczynnik kierunkowy prostej AB: a_AB = (yB - yA) / (xB - xA) = (-3 - 5)/(3 - (-1)) = -2.
3. Wyznacz współczynnik kierunkowy symetralnej (warunek prostopadłości): a_sym = -1 / a_AB = 1/2.
4. Napisz równanie prostej y = (1/2)x + b, przechodzącej przez punkt S(1, 1): 1 = 1/2(1) + b => b = 1/2.
Odpowiedź: y = 0.5x + 0.5.
Zadanie 2. (Styczna do okręgu)
1. Zapisz równanie prostej l w postaci ogólnej: 2x - y - 1 = 0.
2. Promień okręgu r to odległość środka S(3,2) od prostej: r = |2(3) - 1(2) - 1| / √(2² + (-1)²) = |6 - 2 - 1| / √5 = 3/√5 = 3√5/5.
3. Napisz równanie okręgu w postaci kanonicznej: (x - xS)² + (y - yS)² = r².
Odpowiedź: (x - 3)² + (y - 2)² = 9/5.
Zadanie 3. (Romb i wektory)
1. Oblicz długość przekątnej |AC| = √((4-1)² + (2-(-2))²) = √(9 + 16) = 5.
2. Z pola rombu P = (|AC| * |BD|) / 2 wyznacz długość drugiej przekątnej: 25 = (5 * d2)/2 => d2 = 10.
3. Wyznacz środek S przekątnej AC, który jest środkiem okręgu o promieniu r=5 na którym leżą B i D.
4. Wyznacz prostą prostopadłą do AC przechodzącą przez S.
5. Rozwiąż układ równań: prosta prostopadła i okrąg o mieniu S i r=5, wyznaczysz B i D.
Zadanie 4. (Wysokość w trójkącie)
1. Wyznacz równanie ogólne prostej AB przechodzącej przez A(2, 1) i B(6, 5): y = x - 1 => x - y - 1 = 0.
2. Wysokość opuszczona z wierzchołka C to odległość punktu C(-2, 7) od prostej AB.
3. Użyj wzoru na odległość punktu od prostej: d = |-2 - 7 - 1| / √(1² + (-1)²) = |-10| / √2 = 10/√2 = 5√2.
Zadanie 5. (Wektory prostopadłe)
1. Iloczyn skalarny wektorów prostopadłych wynosi zero: u ∘ v = 0.
2. Zapisz równanie: (m-1)(m+1) + (3)(-4) = 0.
3. m² - 1 - 12 = 0 => m² = 13.
Odpowiedź: m = √13 lub m = -√13.
Zadanie 6. (Okrąg z parametrem)
1. Najpierw określ warunek na bycie okręgiem: r² > 0. Zwijając do postaci kanonicznej: (x-2)² + (y+3)² = 13 - k, czyli k < 13.
2. Podstaw y = x do równania okręgu: x² + x² - 4x + 6x + k = 0 => 2x² + 2x + k = 0.
3. Posiada dwa rozwiązania gdy Δ > 0: Δ = 4 - 8k. Rozwiązujemy nierówność 4 - 8k > 0 => k < 1/2.
Odpowiedź: k ∈ (-∞, 1/2).
Zadanie 7. (Wierzchołek trójkąta równobocznego)
1. Trzeci wierzchołek leży na symetralnej odcinka wysokości.
2. Można obrócić wektor AB o 60 oraz -60 stopni lub ułożyć układ równań z długości boków |AC| = |BC| = |AB|.
3. Istnieją 2 rozwiązania odpowiednio symetryczne względem prostej AB.
Zadanie 8. (Kąt między prostymi)
1. Wykorzystaj wzór na kąt ostre z tangensem: tan(α) = |(a1 - a2)/(1 + a1*a2)|.
2. Mamy 1/2 = |(3 - a)/(1 + 3a)|.
3. Rozwiąż dwa równania z wartością bezwzględną: 1 + 3a = 6 - 2a (a=1) LUB -1 - 3a = 6 - 2a (a=-7).
4. Podstawiając a do punktu P(-1,-5) wyznacz b dla każdego z y = ax+b.
Zadanie 9. (Symetria środkowa okręgu)
1. Okrąg wejściowy: S_1 = (2, -3), r = 4.
2. S_1 w symetrii względem punktu S(-1, 1). Ze wzoru ( (x+x')/2 = -1, (y+y')/2 = 1 ).
3. S' = (-4, 5). Promień się nie zmienia r = 4.
Odpowiedź: (x+4)² + (y-5)² = 16.
Zadanie 10. (Okrąg opisany na Δ)
1. Ponieważ trójkąt nie jest przeciwprostokątną na AB, więc na osi OX leży środek O= (x, 0).
2. Z własności okręgu opisanego na trójkącie prostokątnym środek leży na środku przeciwprostokątnej.
3. Odległości |OA| = |OB| muszą być równe. Ułóż rówanie odległości.
4. Rozwiąż x, a potem wyznacz R.
Zadanie 11. (Działania na strzałkach i wektorach)
1. Równanie algebraiczne wektorów: 2w = 3b - a => w = 1.5b - 0.5a.
2. Obliczenia na współrzędnych: w_x = 1.5(-2) - 0.5(3) = -3 - 1.5 = -4.5. w_y = 1.5(4) - 0.5(-1) = 6 + 0.5 = 6.5.
3. Długość wektora: |w| = √((-4.5)² + 6.5²) = √(20.25 + 42.25) = √62.5.
Zadanie 12. (Pole kwadratu i odległość)
1. Oblicz odległość punktu A(1, 1) od przekątnej (którą jest prosta 2x - y - 6 = 0). Wiemy, że A nie należy do tej prostej.
2. Odległość wierzchołka od przekątnej to połowa drugiej przekątnej d/2.
3. d/2 = |2(1) - 1(1) - 6| / √(2² + (-1)²) = 5 / √5 = √5.
4. Pole P = 1/2 * d² = 1/2 * (2√5)² = 1/2 * 20 = 10.
