Schematy rozwiązań dla nauczyciela

Rozwiązanie Zadanie 1.
Warunki: Delta > 0, x1+x2 > 0 i x1*x2 > 0. D = (m+1)^2 - 4m = (m-1)^2. Aby D>0, m != 1. Rozwiązanie to m in (0, 1) U (1, inf).
Rozwiązanie Zadanie 2.
Wsp. a<0. Delta: p^2 + 4p. By nie mialo, D < 0 => p^2 + 4p < 0 => p(p+4) < 0 => p in (-4, 0).
Rozwiązanie Zadanie 3.
Kwadratowe -> k != 0. Jedno to D = 0 => 16 - 4k = 0 => k = 4.
Rozwiązanie Zadanie 4.
Delta = 4m^2 - 4(m^2-1) = 4m^2 - 4m^2 + 4 = 4. Delta > 0, niezależnie od parametru.
Rozwiązanie Zadanie 5.
Samo równanie dzieli się x(mx - m^2)=0. x1=0, wic drugi musi byc ujemny. Drugi to x=m. Czyli dla m < 0.
Rozwiązanie Zadanie 6.
Z x^2 + y^2=1 i y=ax+b zrob kwadratowke (1+a^2)x... Ilosc zalezy tylko w 100% od odleglosci jej od (0,0).
