Zadanie 6.Wymagania: CKE Poziom Rozszerzony (Rownania z wartoscia bezwzgledna)

Dla jakich parametrow p rownanie |x-1|+|x+3|=p nie ma rozwiazania?

Schematy rozwiązań dla nauczyciela

Rozwiązanie Zadanie 1.
Mamy dwa równania: |x-2| - 3 = 5 => |x-2| = 8 => x=10 lub x=-6. Oraz |x-2| - 3 = -5 => |x-2| = -2 (Brak rozw). Odp: x in {-6, 10}.
Rozwiązanie Zadanie 2.
Z definicji: -3 < 2x+5 < 3 => -8 < 2x < -2 => -4 < x < -1.
Rozwiązanie Zadanie 3.
Z trzech przedziałów: ( -inf,0 ): -x - ( -x + 4 ) = x => -4 = x (Akcept). (0, 4): x - ( -x + 4 ) = x => 2x - 4 = x => x=4 (Odrzut domkniety). (4, inf): x - ( x - 4 ) = x => 4 = x (Akcept). Rozwiązania to -4 i 4.
Rozwiązanie Zadanie 4.
Ułóż przedziały x>2 => 1/x + 1/(x-2) = 2... otrzymujemy x = 1 + sqrt(2) lub 1 - sqrt(2) po analizie dopasowań.
Rozwiązanie Zadanie 5.
Moduł to tylko dodatek. |3 - (-4)| + |3 - 4| = |7| + |-1| = 8.
Rozwiązanie Zadanie 6.
Suma odleglosci punktu od -3 i 1 to mininimum 4 (na przedziale <-3,1>). Wiec równanie nie ma rozwiązania dla p < 4.
