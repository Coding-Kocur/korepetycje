Zadanie 6.Wymagania: CKE Poziom Rozszerzony (Wykladnicze i logarytmiczne)

W jakim punkcie wykres f(x) = 2^x - 8 przecina osie?

Schematy rozwiązań dla nauczyciela

Rozwiązanie Zadanie 1.
Podstawy muszą być równe. (2^{-1})^{x-3} > (2^3)^{2x} => 2^{3-x} > 2^{6x}. Z racji podstawy > 1: 3-x > 6x => 7x < 3 => x < 3/7.
Rozwiązanie Zadanie 2.
Dziedzina x > 2. Iloczyn log argumentów: log_2(x(x-2)) = 3 => x^2 - 2x = 8 => x^2 - 2x - 8 = 0. x=4 lub x=-2 (odrzucamy). Odp: x = 4.
Rozwiązanie Zadanie 3.
Wyciągamy 3^{x-1} przed nawias: 3^{x-1}(3 + 1) = 36 => 3^{x-1} = 9 => 3^{x-1} = 3^2 => x-1 = 2 => x = 3.
Rozwiązanie Zadanie 4.
Dziedzina: x > 4. Asymptota pionowa to x = 4 (prawostronna). Nie posiada asymptoty poziomej ze względu na dążenie log do inf.
Rozwiązanie Zadanie 5.
Dziedziina 2x - 1 > 0 => x > 1/2. Obracamy mianownik bo podstawa=0.5: 2x - 1 <= (0.5)^1 => 2x <= 1.5 => x <= 0.75. W dziedzinie: x in (0.5, 0.75>.
Rozwiązanie Zadanie 6.
Dla OX: 2^x - 8 = 0 => 2^x = 8 => x=3. Punkt (3,0). Dla OY: f(0) = 2^0 - 8 = -7. Punkt (0,-7).
