Schematy rozwiązań krok po kroku (dla nauczyciela)

Rozwiązanie Zadanie 1.
Zamieniamy lewą stronę zgodnie ze wzorem \(\cos 2x = \cos^2 x - \sin^2 x\). Zauważ, że mamy odwrotność, więc to \(-\cos 2x = \frac{1}{2} \Rightarrow \cos 2x = -\frac{1}{2}\). Rozwiązujemy \( \cos 2x = -\frac{1}{2} \). Kąt z kosinusem \(-1/2\) to \(\frac{2\pi}{3}\) i \(\frac{4\pi}{3}\). Zejściowe to \(2x = \frac{2\pi}{3} + 2k\pi \Rightarrow x = \frac{\pi}{3} + k\pi\). A drugie \(2x = \frac{4\pi}{3} + 2k\pi \Rightarrow x = \frac{2\pi}{3} + k\pi\). Pod przedział: \(\{\frac{\pi}{3}, \frac{2\pi}{3}, \frac{4\pi}{3}, \frac{5\pi}{3}\}\).
Rozwiązanie Zadanie 2.
Ułamek jest zerowy tylko wtedy, gdy licznik wymosi 0 a mianownik nie. \(\sin x = 0\) daje \(x = k\pi\). Sprawdźmy mianownik: \(1 + \cos x \ne 0 \Rightarrow \cos x \ne -1 \Rightarrow x \ne \pi + 2k\pi\). Co to oznacza? Poodrzuceniu \(x = \pi + 2k\pi\), z odpowiedzi \(x = k\pi\) zostają tylko połówki w zerze: \(x = 2k\pi\).
Rozwiązanie Zadanie 3.
Z definicji dla równań tangensów: \(\tan x + \frac{1}{\tan x} = \frac{4\sqrt{3}}{3} \). Definiujemy zmienną \(t = \tan x \). Równanie \(t^2 - \frac{4\sqrt{3}}{3}t + 1 = 0\). Rozwiazanie delty \(\Delta = \frac{48}{9} - 4 = \frac{12}{9} \). Wiec \(\sqrt{\Delta} = \frac{2\sqrt{3}}{3} \). Mamy \(t_1 = \frac{1}{\sqrt{3}} = \frac{\sqrt{3}}{3}\) Oraz \(t_2 = \sqrt{3}\). Tango warte \(\frac{\sqrt{3}}{3}\) to \(\frac{\pi}{6} + k\pi\). Dla \(\sqrt{3}\) to \(\frac{\pi}{3} + k\pi\). Odp to te serie.
Rozwiązanie Zadanie 4.
To wzór po lwej stronie: \( \sin(2x) = 1 \). Wtedy \(2x = \frac{\pi}{2} + 2k\pi \Rightarrow x = \frac{\pi}{4} + k\pi \). Sprawdzamy dla naszego przedziału: dla k=0, \(x = \pi/4\). Dla k=-1, \(x = -3\pi/4\). Pozostałe wychodzą poza domknięcia przedziału.
Rozwiązanie Zadanie 5.
Musimy sprowadzić do jednej funkcji. Jedynka trygonometryczna: \(\cos^2 x = 1 - \sin^2 x\). Zatem \(2(1-\sin^2 x) + 3\sin x - 3 = 0 \Rightarrow -2\sin^2 x + 3\sin x - 1 = 0\). Zmieniamy t=\(\sin x\). \(-2t^2+3t-1=0\). Wtedy \(\Delta = 9 - 8 = 1\). Zatem \(t_1 = 1/2\), \(t_2 = 1\). Obie liczby leżą w \(\langle -1,1 \rangle\). Równanie ma rozwiązania: \( \sin x = 1 \Rightarrow x = \pi/2 + 2k\pi\). Oraz \(\sin x = 1/2 \Rightarrow x = \pi/6 + 2k\pi\) lub \(x = 5\pi/6 + 2k\pi \).
Rozwiązanie Zadanie 6.
Zadanie pułapka maturalne CKE. Wartość bezwzględna lewej strony daje zawsze wartość liczbową \( \ge 0 \). Ponieważ prawa strona jest ujemna, równanie jest tożsamościowo sprzeczne. Brak rozwiązań \( x \in \emptyset \) .
