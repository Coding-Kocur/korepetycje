Dział CKE: Dowody algebraiczne

Zestaw 6 starannie dobranych zadań ze wskazówkami i rozwiązaniami, opracowanych pod kątem wymagań nowej matury CKE na poziomie rozszerzonym.

Zadanie 1.Wymagania: CKE Poziom Rozszerzony (Dowody algebraiczne)

Wykaż, że dla dowolnych liczb dodatnich a, b zachodzi a/b + b/a >= 2. (Dowód - CKE 2017)

Zadanie 2.Wymagania: CKE Poziom Rozszerzony (Dowody algebraiczne)

Udowodnij, że jeśli x+y = 3, to x^2 + y^2 >= 4.5.

Zadanie 3.Wymagania: CKE Poziom Rozszerzony (Dowody algebraiczne)

Wykaż, że dla całkowitych k, wyrażenie k(k+1)(k+2)(k+3) + 1 jest kwadratem liczby całkowitej. (CKE 2019)

Zadanie 4.Wymagania: CKE Poziom Rozszerzony (Dowody algebraiczne)

Wykaż, że liczba 3^90 + 5^90 jest podzielna przez 34.

Zadanie 5.Wymagania: CKE Poziom Rozszerzony (Dowody algebraiczne)

Wykaż założenie p > 0, q > 0 ... p^3 + q^3 >= pq(p+q).

Zadanie 6.Wymagania: CKE Poziom Rozszerzony (Dowody algebraiczne)

Dowieść, że suma trzech kolejnych sześcianów liczb naturalnych jest podzielna przez 9.

Schematy rozwiązań dla nauczyciela

Rozwiązanie Zadanie 1.
Mnożąc obustronnie przez dodatnie ab, mamy a^2 + b^2 >= 2ab, czyli (a-b)^2 >= 0, co jest zawsze prawdziwe.
Rozwiązanie Zadanie 2.
Z a^2 + b^2 >= (a+b)^2 / 2 podstawiamy 3^2 / 2 = 9/2 = 4.5. Koniec dowodu.
Rozwiązanie Zadanie 3.
Wymnażamy skrajne z środkowymi: (k^2+3k)(k^2+3k+2)+1. Podstawmy t = k^2+3k. Mamy t(t+2)+1 = t^2+2t+1 = (t+1)^2. C.n.d.
Rozwiązanie Zadanie 4.
Z definicji podzielności sum wielomianów (a^n + b^n), skoro n jest parzyste, należy sprowadzić to do kwadratów (3^2)^45 + (5^2)^45 = 9^45 + 25^45. Przy nieparzystej potędze 45, jest to podzielne przez (9+25) = 34.
Rozwiązanie Zadanie 5.
Odejmując prawa stronę, wyłączamy (p+q)(p^2 - pq + q^2) - pq(p+q) >= 0. Wyciągamy (p+q) a w naiwasie zostaje (p-q)^2.
Rozwiązanie Zadanie 6.
(n-1)^3 + n^3 + (n+1)^3 = 3n^3 + 6n = 3n(n^2+2). Analiza z reszto modulo 3 pokazuje, że zawsze to wyrażenie posiada czynnik dodający podzielność przez 9.
